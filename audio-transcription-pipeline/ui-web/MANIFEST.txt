API ENDPOINTS
-------------

REST API:
  POST   /api/upload                      - Upload audio file
  GET    /api/transcriptions/{job_id}     - Get transcription result
  GET    /api/transcriptions/{job_id}/status - Get job status
  GET    /api/transcriptions              - List all transcriptions
  DELETE /api/transcriptions/{job_id}     - Delete transcription

WebSocket:
  WS     /ws/transcription/{job_id}       - Real-time progress updates

QUICK START
-----------

Step 1: Run setup script
  cd ui-web
  ./scripts/setup-local.sh

Step 2: Add API keys to backend/.env
  OPENAI_API_KEY=your_key
  HUGGINGFACE_TOKEN=your_token

Step 3: Start backend (Terminal 1)
  cd backend
  source venv/bin/activate
  python -m app.main

Step 4: Start frontend (Terminal 2)
  cd frontend
  npm run dev

Step 5: Open browser
  http://localhost:5173
