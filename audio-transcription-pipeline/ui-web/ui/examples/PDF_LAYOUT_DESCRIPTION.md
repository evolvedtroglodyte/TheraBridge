## Color Coding

### Speaker Headers (Colored Backgrounds)
- **Therapist (SPEAKER_00):** Blue background (#3B82F6) with white text
- **Client (SPEAKER_01):** Green background (#10B981) with white text
- **Additional Speakers:** Amber, Red, Purple, Pink backgrounds

### Text Styling
- **Title:** 24pt, Bold, Indigo (#4F46E5)
- **Section Headers:** 14pt, Bold, Indigo (#4F46E5)
- **Body Text:** 10pt, Regular, Black
- **Timestamps:** 10pt, Regular, Gray (#6B7280)
- **Metadata Labels:** 10pt, Bold, Black
- **Metadata Values:** 10pt, Regular, Black

---

## Page Layout Specifications

### Dimensions
- **Page Size:** US Letter (8.5" x 11")
- **Margins:** 20mm all sides
- **Content Width:** 170mm (6.69")

### Typography
- **Font Family:** Helvetica (sans-serif)
- **Line Spacing:** 1.2x (body text)
- **Paragraph Spacing:** 4-6pt between segments

### Visual Elements
- **Header Line:** 0.5pt gray horizontal line under title
- **Speaker Boxes:** Rounded corners (2pt radius), colored background
- **Statistics Table:** Light gray background (#F5F5F5), 1pt borders
- **Page Numbers:** Centered, bottom of page, 9pt gray text

---

## Page Break Logic

PDF automatically inserts page breaks when:
1. A speaker segment would overflow the page (keeps segments intact)
2. Less than 40mm of space remains at bottom of page
3. Section headers require new page for readability

**Result:** Clean, professional layout with no orphaned content


- High contrast text (black on white for body)
- Legible font sizes (10pt minimum)

### Print Optimization
- Standard US Letter size (compatible with all printers)
- 300 DPI equivalent quality
- Color-safe (looks good in grayscale)
- Single-sided layout

## Example Use Cases

### Clinical Documentation
```
Therapist exports session transcript as PDF
→ Reviews for accuracy
→ Adds clinical notes
→ Stores in patient's electronic health record (EHR)
→ Shares with supervisor for case consultation
```

### Research Analysis
```
Researcher exports multiple sessions as CSV
→ Imports into R or Python
→ Analyzes turn-taking patterns
→ Calculates speaking time ratios
→ Generates visualization graphs
```

### Quality Assurance
```
QA team exports test session as PDF
→ Compares with manual transcript
→ Highlights discrepancies
→ Reports accuracy metrics
→ Archives for compliance
```
